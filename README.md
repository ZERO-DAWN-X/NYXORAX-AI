## Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/ZERO-DAWN-X/NyxoraX.git

# Navigate to project directory
cd NyxoraX/vscode

# Install dependencies
npm install

# Build the editor
npm run compile

# Launch NyxoraX
npm run watch
```

### System Requirements

- **Node.js** 16.0 or higher
- **npm** 7.0 or higher
- **Git** for version control
- **4GB RAM** minimum, 8GB recommended
