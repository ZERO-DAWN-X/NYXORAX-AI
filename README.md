## ⚡ Quick Start

<div align="center">

### 🛠️ Installation & Setup

</div>

```bash
# 📥 Clone the repository
git clone https://github.com/ZERO-DAWN-X/NyxoraX.git

# 📂 Navigate to project
cd NyxoraX/vscode

# 📦 Install dependencies
npm install

# 🔨 Build the editor
npm run compile

# 🚀 Launch NyxoraX
npm run watch
```
